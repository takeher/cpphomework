树的层次遍历可以使用广度优先搜索实现。
从根节点开始搜索，每次遍历同一层的全部节点，使用一个列表存储该层的节点值。
如果要求从上到下输出每一层的节点值，
做法是很直观的，在遍历完一层节点之后，
将存储该层节点值的列表添加到结果列表的尾部。
这道题要求从下到上输出每一层的节点值，
只要对上述操作稍作修改即可：
在遍历完一层节点之后，将存储该层节点值的列表添加到结果列表的头部。
为了降低在结果列表的头部添加一层节点值的列表的时间复杂度，
结果列表可以使用链表的结构，
在链表头部添加一层节点值的列表的时间复杂度是O(1)。
我们需要返回一个vector或list，
它不方便在头部插入元素（会增加时间开销），
所以我们可以先用尾部插入的方法得到从上到下的层次遍历列表，然后再进行反转。
